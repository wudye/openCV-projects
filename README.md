反向传播（Backpropagation）
1. 误差计算： 首先，计算模型在当前迭代中的预测输出与真实目标之间的差距，这通常转化为对损失函数（如交叉熵、均方误差等）的计算。
2. 梯度计算： 使用链式法则从输出层开始，按相反的顺序逐层计算每个中间变量和参数对于总损失函数的梯度。也就是说，计算损失相对于网络中每层权重和偏置的导数。
3. 参数更新： 根据计算出的梯度，利用优化算法（如梯度下降法、动量法、Adam等）来更新网络的权重和偏置。更新的方向与梯度方向相反，目的是减少损失函数的值。


1. 损失函数 (Cross-Entropy)：裁判员
你提到的交叉熵（Cross-Entropy）常用于分类任务。
它的职责：衡量“概率分布”的差异。比如模型预测图片是猫的概率是 0.7，而真实标签是 1（100%是猫），交叉熵就会计算这两个数值之间的距离。
为什么用它？ 相比简单的减法，交叉熵在误差较大时能产生更大的梯度，让模型在早期“学得更快”。
2. 反向传播 (Backpropagation)：传声筒/翻译官
它是连接“误差”和“参数”的桥梁。
它的职责：通过微积分的链式法则，把损失函数算出的总误差，拆解并分配给网络中成千上万个参数（权重 
 和偏置 
）。
核心产出：它计算出的是梯度（Gradient）。梯度告诉我们：如果要让误差下降，每个参数应该往哪个方向调（加还是减）。
3. Adam 优化器：经验丰富的舵手
反向传播只给出了“方向”，但真正去操纵方向盘、踩油门的是 Adam。
自适应学习率：这是 Adam 的杀手锏。传统的优化器（如 SGD）所有参数共用一个速度（学习率）。而 Adam 会给每个参数“量体裁衣”：
更新频繁的参数：说明它方向比较乱，Adam 会减小它的步长。
更新稀疏的参数：说明它需要多走走，Adam 会加大它的步长。
动量机制：Adam 还会参考之前的更新方向，防止在寻找最优解的过程中产生剧烈的震荡。

Adam 算法的核心在于为每个参数计算自适应学习率。

初始时，dam 初始化两个向量 m 和 v，它们的形状与模型参数 θ 相同。向量 m 用于存储梯度的移动平均值，而 v 则跟踪梯度平方的移动平均值。这些移动平均值对于 Adam 的自适应调整至关重要。还初始化了一个时间步计数器 t，初始值为零，它跟踪算法完成的迭代（或更新）次数。


优化器则将损失
函数计算得到的误差，通过反向传播作用于神经网络内部的可训练参
